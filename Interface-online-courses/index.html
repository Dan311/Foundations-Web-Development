Do I need experience?
For the Basics of Web Development course, no - this course will teach you everything you need to know. For the Mobile Web Development Course, yes - you should have a solid understanding of both HTML & CSS, included uing advanced selectors like nth-of-type and :target.
Do I need to install any software?
You only need Chrome - everything will be presente online, and you will use our online development platform for all coding work.
Is this course hard?
Maybe - kinda depends on you! The material is presented in a very step-by-step manner asit guides you through new topics while you are devloping a real website. Remember that these are also self-paced courses, so if you need to review a setion multiple times to really "get it," you can.
How long is the course?
Courses are open from the first of the month for 31 days, and each course is divided into about 8-10 sections, so you typically have about 3 days on average to complete each section. Each section can take anywhere from 1-4 hourse to complete.
